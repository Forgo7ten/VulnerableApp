# VulnerableApp

Android应用层漏洞靶场，总结和复现一下自己学习过的漏洞。

同时为了锻炼Android开发，采用了kotlin语言，编写了简单而丑陋且无用的界面😭。


## 漏洞列表

> 目前还在持续更新中，之后会按照漏洞类型做一个分类，StarStar~

1. Manifest中定义组件未实现漏洞
2. 应用数据备份配置不当漏洞
3. 应用调试模式配置不当漏洞
4. App通用型拒绝服务漏洞
5. ContentProviderURI授权不当漏洞
